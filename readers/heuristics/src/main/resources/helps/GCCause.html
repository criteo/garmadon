Indicates GC causes that should not be triggered on a correctly sized application

Causes

  • Metadata GC Threshold
  • Ergonomics

How to resolve?

  • Metadata GC Threshold: Set/Increase metaspace size with the following JVM options: -XX:MetaspaceSize=128M
  • Ergonomics: Set/Increase initial heap space with the following JVM options: -Xms1G
